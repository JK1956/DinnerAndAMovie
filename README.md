# DinnerAndAMovie
An app that gives you a random meal and movie based on the food &amp; genre you desire! 


## Technologies
 * HTML
 * CSS / CSS FRAMEWORK
 * JAVASCRIPT
 * [MovieDB API](https://www.themoviedb.org/ "MovieDB Homepage")
 * [TheMealDB API](https://www.themealdb.com/api.php "MealDB Homepage")
